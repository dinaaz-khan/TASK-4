# 🧠 Quiz Master App

A responsive and interactive quiz application built using **HTML, CSS, and JavaScript**.  
This project allows users to answer multiple-choice questions, track progress, and view their final score with feedback.

---

## 📌 Overview

The **Quiz Master App** is a frontend project designed to simulate a real-world quiz system.  
It includes dynamic question rendering, answer validation, progress tracking, and result analysis.

---

## 🚀 Features

- ✅ Dynamic question loading
- 📊 Progress bar indicator
- 🔁 Previous & Next navigation
- ⚠️ Validation for unanswered questions
- 🧮 Score calculation
- 📈 Percentage display
- 💬 Performance-based feedback
- 🔄 Restart quiz functionality
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & Layout  
- **JavaScript (ES6)** – Logic & Interactivity  

---

## ⚙️ How It Works

1. The quiz loads with a set of predefined questions.
2. User selects an answer for each question.
3. Navigation buttons allow moving between questions.
4. Progress bar updates dynamically.
5. Final score and feedback are shown after submission.
